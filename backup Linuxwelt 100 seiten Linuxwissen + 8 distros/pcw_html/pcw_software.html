Unetbootin 6.55 Linux-Version (64/32 Bit)

Mit Unetbootin schieben Sie den Inhalt einer bootfähigen ISO-Datei von Ubuntu und seinen Varianten auf eine Speicherkarte oder einen Stick und starten den PC oder das Notebook dann von diesem Datenträger. In der aktuellen Version 6.25 hat sich Unetbootin in zwei separate Versionen für 32-Bit und 64-Bit-Systeme aufgeteilt. Die Binärdatei unetbootin-linux64-655.bin funktioniert unter 64-Bit-Linux-Systemen, während unetbootin-linux-655.bin für 32-Bit-Systeme gemacht ist. Das Tool steht unter der GNU General Public Licence 2.

Mehr Infos zu dem Tool gibt es auch unter http://unetbootin.sourceforge.net/. Eine Schritt-für-Schritt-Anleitung dazu finden Sie hier auf Heft-DVD.


Unetbootin 6.55 Windows-Version (für alle Versionen)

Praktischerweise liegt das Tool Unetbootin auch für Windows vor. Sie finden diese Version hier auf Heft-DVD (unetbootin-windows-655.exe). Sie transferieren damit ISO-Dateien von Linux-Distributionen, wie beispielsweise Ubuntu unter Windows auf USB-Sticks oder Flash-Speicherkarten. Das Programm funktioniert unter allen Windows-Versionen ab Windows Vista. Unetbootin steht unter der GNU General Public Licence 2.

Mehr Infos zu dem Tool gibt es auch unter http://unetbootin.sourceforge.net/.


Unetbootin 6.55 Mac-OS-X-Version

Das Programm Unetbootin liegt jetzt auch in einer Version für Mac OS X vor. Sie finden diese Version als unetbootin-mac-655.dmg auf Heft-DVD. Sie transferieren damit ISO-Dateien diverser Distributionen unter Mac OS X auf USB-Sticks oder Flash-Speicherkarten. Das Tool steht unter der GNU General Public Licence 2.

Mehr Infos zu dem Tool gibt es auch unter http://unetbootin.sourceforge.net/.


Imgburn 2.5.8.0

Das kostenlose Tool für Windows XP/Vista/7/8 brennt Abbild-Dateien von CDs und DVDs. Dabei versteht sich die Software mit allen verbreiteten Image-Formaten wie ISO, BIN und NRG. Es liegt als EXE-Datei SetupImgBurn_2.5.8.0.exe auf Heft-DVD.
Hinweis: Die Freeware ist werbefinanziert und bietet bei der Installation die optionale Einrichtung von Werbe-Links auf dem Desktop an, sowie die Browser-Toolbar von Ask.com. Diese Optionen lassen sich natürlich auch bei der Installation de-aktivieren - nur übersehen darf man dies nicht.

Mehr Infos unter http://www.imgburn.com


Win32 Disk Imager 1.0 Binary

Der Disk Imager für Windows liegt als Win32DiskImager-1.0.0-binary.zip auf Heft-DVD. Das Programm kopiert IMG-Dateien sowie ISO-Dateien (Für USB und CD/DVD), in der die meisten Linux-Distributionen vorliegen, auf USB-Sticks oder Flash-Speicherkarte. Das Open-Source-Programm (GPL2) funktioniert unter allen Windows-Versionen ab Windows Vista und verlangt keine Installation.

Mehr Infos unter https://wiki.ubuntu.com/Win32DiskImager


Win32 Disk Imager 1.0 Installer

Der Disk Imager für Windows kopiert IMG-Dateien und hybride ISO-Dateien (Für USB und CD/DVD), in der beispielsweise die meisten Linux-Distributionen vorliegen auf USB-Sticks oder Flash-Speicherkarte. Das Open-Source-Programm (GPL 2) funktioniert unter allen Windows-Versionen ab Windows Vista. Es liegt als ZIP-Datei win32diskimager-1.0.0-install.exe auf Heft-DVD.

Mehr Infos unter https://wiki.ubuntu.com/Win32DiskImager


Putty 0.70 (64/32 Bit)

Seit Jahren ist Putty der populärste SSH-Client für Windows-Systeme, da Microsoft von Haus aus kein Terminal für SSH-Verbindungen zu Linux/Unix/BSD-Servern mitliefert. Es handelt sich um eine MSI-Datei für Windows, die sich über einen Rechtsklick installieren lässt. Das Programm ist englischsprachig und steht als Open-Source-Software unter der freizügigen MIT-Lizenz. Als 32-Bit-Programm liegt Putty als putty-0.70-installer.msi und in der 64-Bit-Variante als putty-64bit-0.70-installer.msi auf Heft-DVD.

Mehr Infos unter http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html


Kitty 0.70.0.2

Als Abspaltung des originalen Putty kann Kitty mit einigen Zusatzfunktionen aufwarten, welche die Arbeit mit dem SSH-Client komfortabler machen. So gibt es Login-Scripts, Kommandokürzel, eine übersichtlichere Session-Verwaltung und die Möglichkeit, Kitty in den Systray zu verkleinern. Das Programm ist englischsprachig und steht unter der MIT-Lizenz. Es liegt als ausführbare EXE-Datei kitty.exe auf Heft-DVD und verlangt keine Installation.

Mehr Infos unter http://www.9bis.net/kitty/index.php.
